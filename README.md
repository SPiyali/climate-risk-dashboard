# 🌍 Climate Risk Intelligence Dashboard

An interactive data-driven dashboard to analyze climate risks across different regions based on environmental and population factors.

🔗 **Live App:**  
https://climate-risk-dashboard-dj5wmqqe9app2zj6vj2trop.streamlit.app/

---

## 📌 Overview

The Climate Risk Intelligence Dashboard helps users explore climate-related risks using real-world datasets.  
It provides insights based on temperature, rainfall, and population trends using interactive visualizations.

---

## ⚙️ Tech Stack

- Python 🐍
- Streamlit 🚀
- Pandas 📊
- NumPy 🔢
- Plotly 📈

---

## ✨ Features

- 📊 Interactive data visualization using Plotly  
- 🌦️ Climate risk analysis (temperature, rainfall, population)  
- 🌍 Country-wise comparison  
- ⚡ Real-time dashboard interface  
- 📉 Risk scoring system for identifying high-risk regions
